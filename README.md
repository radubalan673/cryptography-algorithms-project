# cryptography-algorithms-project

## Description
This project contains Python implementations of classical and modern cryptographic algorithms for educational purposes.

## Classical Ciphers
- Affine Cipher
- Caesar Cipher
- Polybius Cipher
- Trithemius Cipher

## Modern Algorithms
- AES
- ElGamal
- TEA 
- XTEA

## Goal
The aim of this project is to understand cryptographic principles through practical implementation.

## Technologies
- Python
